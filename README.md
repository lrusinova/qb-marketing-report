# QB Marketing · H2 2026 Program Report

Live program health dashboard for the QuickBase Marketing team.

## What's in here

- **Program Health** — RAG status across all 7 H2 strategic initiatives
- **Output Tracker** — published content by channel and BU (pulled from Jira Done status)
- **Roadmap** — Gantt view of all programs across Q2 · Q3 · Q4 2026
- **By BU** — FastField / Pave / Core breakdown

## How it's updated

Data is pulled live from Jira MKT and regenerated by Claude. To refresh, ask Claude to "run the weekly report" — it queries Jira and produces an updated `index.html`.
